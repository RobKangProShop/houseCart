Cloud sync (GitHub Gist)

Free cross-device sync via a secret GitHub Gist. Every save auto-pushes; on launch the latest version is pulled. Each push is a git commit — you get free version history.

How to set this up (one time, ~2 min)
  1. Important: use a classic token, not fine-grained. (GitHub's fine-grained PATs cannot access the Gist API — they return 403.)
  2. Go to github.com/settings/tokens/new
  3. Note: HouseCart sync · Expiration: 90 days
  4. Under Select scopes, check only gist (under "Create gists"). Leave everything else unchecked.
  5. Click Generate token and copy the ghp_… value (shown only once).
  6. Paste it below. Leave Gist ID blank — the first save creates a private Gist automatically.
  7. On your other devices, paste the same token and the same Gist ID (visible here after the first sync).

⚠️ Token is stored unencrypted in this browser's localStorage. Anyone with this browser profile can read it. Use a token scoped only to Gists and rotate it periodically.
